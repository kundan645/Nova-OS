# 🚀 NovaOS – The AI Operating System for Businesses

> **One AI. One Dashboard. Complete Business Control.**

NovaOS is an AI-powered business operating system that helps shop owners and businesses manage their daily operations from a single dashboard. Instead of switching between Excel, WhatsApp, accounting software, CRM tools, invoices, and reports, NovaOS brings everything together with an intelligent AI assistant.

---

## 🌟 Problem

Small and medium businesses use multiple disconnected tools to manage their business:

- 📊 Excel for sales
- 💬 WhatsApp for customers
- 📁 Google Drive for documents
- 📑 Bills & invoices
- 💰 Accounting software
- 📦 Inventory systems

As a result,

- Information is scattered.
- Reports take time.
- Business owners cannot make quick decisions.
- Customer follow-ups are missed.
- Manual work increases.

---

# 💡 Solution

NovaOS acts as an **AI Employee** for your business.

It understands your documents, analyzes your sales, tracks payments, monitors inventory, answers questions, and provides smart business insights—all from one dashboard.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/novaos.git
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

---

# ⚙ Environment Variables

Create a `.env` file.

```env
VITE_SUPABASE_URL=your_project_url

VITE_SUPABASE_ANON_KEY=your_anon_key
```
